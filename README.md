# Personal Portfolio

A clean, minimal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Quick Start

1. Open `index.html` in your browser, or run a simple local server:
   ```bash
   # Using Python
   python3 -m http.server 8000

   # Using Node (npx)
   npx serve
   ```
2. Visit `http://localhost:8000`

## Customization

- **Your Name** — Replace "Your Name" in the nav logo, footer, and page title
- **Hero** — Update the tagline, title, subtitle in the hero section
- **About** — Edit the about paragraph to tell your story
- **Projects** — Add your real projects: titles, descriptions, links, and replace the placeholder images
- **Contact** — Update email, LinkedIn, GitHub, Twitter links
- **Colors** — Edit CSS variables in `styles.css` (`:root`) to change the accent color and palette

## Structure

- `index.html` — Semantic markup with Hero, About, Work, Contact
- `styles.css` — Responsive layout, typography, and animations
- `main.js` — Header scroll effect and mobile menu toggle
